### Conception du site : 
- Méthode Crazy 8 pour le design, 
- Utilisation de Figma pour le design, 
- Mise à jour tout au long du projet via la méthode Agile. 

### Organisation : 
- Mise en place d’un repo Github, 
- Création d’une convention d'homogénéisation. 

### Soft Skills : 
- Travail en équipe, 
- Travail autonome, 
- Organisation via GitHub / gestion des conflits, 
- Travail avec une limite de temps
</div>
<div>
### Méthode agile Scrum : 
- Daily Stand up, 
- Gestion du Backlog/UserStories, 
- Création des tickets sous Trello, 
- Planification des sprints par semaine, 
- Réalisation de sprint review, 
- Adaptation de la stratégie en fonction des problématiques rencontrées. 

### Partie technique : 
- Réalisation d'un site Frondend avec quelque appelle api 
- Utilisation de leaflet pour aficher une carte 
- Utilisation de L'API Pole Emploi pour récupérer les offres d'emplois de la Region 
- Utilisation de L'API de Toulouse Metropole pour Geolocaliser des Adresse
- Deployer le site sur Netlify
</div>
